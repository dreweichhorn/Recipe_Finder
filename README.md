🍳 Recipe Finder App
A sleek, responsive web application that leverages TheMealDB API to automate the recipe discovery process. This tool helps users search for culinary inspiration and generate comprehensive cooking guides, complete with ingredient lists and video tutorials in seconds.

🚀 Features:
Keyword Search: Input a meal name or main ingredient to receive a variety of global recipe suggestions.

Detailed Recipe Generation: Instantly view structured cooking instructions with a clear introduction, body, and conclusion.

Ingredient & Measure Mapping: Automatically parses and displays precise measurements for every ingredient required.

Multimedia Integration: Includes direct links to YouTube tutorials to provide a visual walkthrough for complex dishes.

Mobile-First Design: Optimized with CSS Grid and Flexbox to ensure a seamless experience across all devices.

🛠️ Tech Stack:
Frontend: HTML5, CSS3 (using CSS Variables and Grid)

Logic Framework: Vanilla JavaScript (ES6+)

External API: TheMealDB JSON API

Iconography: Font Awesome 7.0.1

📋 Prerequisites
Before running the app, ensure you have:

A modern web browser (Chrome, Firefox, or Safari).

An active internet connection to fetch data from the API.

🏃 How to Run
Simply open the index.html file in your browser, or if you are using VS Code, use the Live Server extension to launch the application at http://127.0.0.1:5500.
